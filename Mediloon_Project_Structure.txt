# ğŸ”Œ API ENDPOINTS

All endpoints are prefixed with `/api`.

## Authentication
- POST /auth/register
- POST /auth/login
- GET /auth/me

## Medicines
- GET /medicines
- GET /medicines/:id
- POST /medicines
- PUT /medicines/:id
- DELETE /medicines/:id

## Inventory
- GET /inventory
- PUT /inventory/update
- GET /inventory/alerts

## Orders
- POST /orders
- GET /orders/my
- GET /orders
- GET /orders/:id

## AI & Agentic
- POST /ai/order
- POST /ai/chat
- GET /ai/predictions/:userId
- GET /ai/logs
